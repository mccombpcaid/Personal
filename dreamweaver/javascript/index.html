jQuery FAQs

What is jQuery

jQuery is a library of the JavaScript functions that you're most likely to need as you develop websites.

Why is jQuery becoming so popular?

Three reasons:

  • It's free.
  • It lets you get more done in less time.
  • All of it function are cross-broswer compatible.

Which is harder to learn: jQuery of JavaScript?

For most function, jQuery is significantly easier to lern and use than JavaScript. But remember: jQuery is JavaScript.
